# 数学题



已知 $a, b, c ∈R^+$ 且满足$\frac{1}{a} + \frac{1}{2b}+ \frac{1}{3c}=1$ 求证: $a+2b+3c≥9$

> [!Note]
>
> 解题思路: 使用柯西不等式
>
> $(a^2+b^2)(c^2+d^2)≥(ac+bd)^2$
>
> ~~$(a+b)^2≥a^2+b^2$~~  (不对)
>
> 
>
> $(a+b)(a+b)≥aa+bb≥\sqrt{}$ 
>
> $(a+b)(c+d)≥ac+bd$
>
> $(a^2+b^2)(c^2+d^2)≥(ac+bd)^2≥{(ac)}^2+{(bd)}^2$

$a+2b+3c=(a+2b+3c)\cdot(\frac{1}{a} + \frac{1}{2b}+ \frac{1}{3c})$

$=(({\sqrt{a}})^2+({\sqrt{2b}})^2+({\sqrt{3c}})^2)(({\frac{1}{\sqrt{a}}})^2+({\frac{1}{\sqrt{2b}}})^2+({\frac{1}{\sqrt{3c}}})^2)$ 

$≥(1+ 1 + 1)^2=9$

## 快速估算根号

对于任意数x 有 $x=a^2+b$ ,  $x-a^2 = b$ -> ${\sqrt{x}}^2-a^2=b$ -> $(\sqrt{x}+a)\cdot(\sqrt{x}-a)=b$  

-> ${\sqrt{x}-a} = \frac{b}{\sqrt{x}+a}$ 

-> $\sqrt{x} = a+\frac{b}{\sqrt{x}+a}$ 

-> $\sqrt{x} = a+\frac{b}{\sqrt{x}+a} = a+\frac{b}{{a+\frac{b}{\sqrt{x}+a}}+a}$

-> $\sqrt{x}=a+\frac{b}{2a+\frac{b}{\sqrt{x}+a}}=a+\frac{b}{2a+\frac{b}{2a+\frac{b}{\sqrt{x}+a}}}$
