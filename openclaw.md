# OpenClaw





## 特性

- 自主运行, 运行在本地
- 能帮你干活







## 核心

- LLM

- LLM Agent

- 网关(控制访问层)

    - 记忆(Memory)
    - 技能(Skills)
        - 操作流程, 规范, 注意事项
        - 自动化办公, 创作, 编码等
    - 工具(Tools)
        - 外部功能支持

- 插件(plugin)

    - 能力扩展

- 通讯软件接入(聊天控制, 频道隔离)

    - 钉钉
    - 飞书
